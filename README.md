---
  有個方便管理、不用另開專案的方法是：
  1. 進入在GitHub放作業的Repositories頁面  
  2. 選擇Settings分頁  
  3. 往下拉到GitHub Pages段落，  
     將Source由None改為master branch，儲存  
     ![範例圖](img/gitpage.gif)  
  4. 之後就可以直接用下面的網址交作業：  
     `https://帳號.github.io/放作業的專案名/檔案名.副檔名`
